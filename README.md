# Quora-mini-project
This is a mini project of online knowledge sharing platform .

# Mini Quora Clone
A simple Quora-inspired web application where users can create, read, update, and delete posts (CRUD-Operations).

# Features
Create a new question/post
View all questions/posts
Edit existing posts
Delete posts
Simple and user-friendly interface --
# Languages Used :
HTML
CSS
JavaScript
Node.js
Express.js
EJS (if used)
CRUD Operations
# Create
Users can add a new question/post.

# Read
Users can view all available questions/posts.

# Update
Users can edit their existing questions/posts.

# Delete
Users can remove questions/posts.
